编译设置开发环境

  • 需要安装MongoDB
  1. 获取EOS源码及其所有子模块 git clone https://github.com/eosio/eos --recursive
  • 如果克隆代码时未带–recursive参数，那么可切换到项目路径下面执行如下命令，更新子模块 git submodule update --init --recursive
    1. cd eos;./eosio_build.sh darwin full // mac
    2. cd eos;./eosio_build.sh //linux
    3. cd build;sudo make install
    4. 启动nodeos ./nodeos -e -p eosio --plugin eosio::chain_api_plugin --plugin eosio::history_api_plugin(keosd会自启动)

操作

  1. 创建默认钱包 ./cleos wallet create（加上–to-console 将钱包密码打印到窗口）
  2. 创建一般的钱包 ./cleos wallet create -n XXX // xxx为钱包名称
  3. 查看已创建的钱包 ./cleos wallet list//钱包名称后的*表示解锁状态
  4. 给指定钱包上锁 ./cleos wallet lock -n xxx
  5. 解锁指定钱包 /cleos wallet unlock -n xxx --password xxxxx
  6. 创建秘钥 ./cleos create key ./cleos import -n 钱包名称 + 私钥
  1. 创建秘钥 cleos wallet create_key -n 钱包名 此命令等价于上面两个命令
  2. 创建账户 ./cleos create account ${authorizing_account} ${new_account} ${owner_key} ${active_key}
  3. 查看账户信息 ./cleos get account -j ${accountName}

  4. EOS中，有五个基础的智能合约，分别是eosio.bios、eosio.token(代币合约)、 exchange和eosio.msig(用于多用户签名和用户权限管理)、eosio.system(用户投票、将用户注册成为生产者)
